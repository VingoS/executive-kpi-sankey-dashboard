# Executive KPI Sankey Dashboard

Visualização de indicadores financeiros e demonstrações de resultados através de gráficos Sankey interativos desenvolvidos em Python.

## Objetivo

Transformar dados financeiros em fluxos visuais que facilitem a análise executiva e a tomada de decisão.

O projeto busca representar visualmente a movimentação de valor entre as principais linhas da demonstração de resultados.

## Indicadores Iniciais

- Receita Bruta
- Deduções
- Receita Líquida
- Custos
- Margem Bruta
- Despesas Operacionais
- EBITDA
- Resultado Financeiro
- Lucro Líquido

## Tecnologias

- Python
- Pandas
- NumPy
- Plotly
